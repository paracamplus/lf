Conclusion

When the page is loaded, code runs on both side but, if you return to `first page` (with the button) then back again to `first test`. Only the client code is re-run. Server code is only run when the page is initially loaded or re-loaded that is, when really requested from the server. The `first page` button is processed on the client only.
