Experiments around evaluation order in Svelte Sapper

To run these tests and analyze them, I use the following setup:

  • a terminal running npm run dev in order to see the console.log() output from the server.
  • a browser aiming at http://localhost:3000/tests
  • the Web console of the browser in order to see the console.log() output produced by client code.
  • an editor to look at the source of the tests. You may alternately browse the GitHub repository.
